# Arduino Course

**By the end of this course, you'll be able to buy an Arduino starter kit and build your own simple project.**

---

## [Week 1 — Outputs and Programming Basics](Week1/README.md)

**Goal:** Make the Arduino do something.

- Electronics fundamentals, board tour, and the IDE
- Digital output — controlling LEDs from code

> Software can control hardware.

---

## [Week 2 — Inputs and Interaction](Week2/README.md)

**Goal:** Make hardware respond to the world.

- Digital input (button) and analog input (potentiometer, photoresistor)
- Serial Monitor and thresholds

> Hardware can sense the world.

---

## [Week 3 — Build the Night Light](Week3/README.md)

**Goal:** Assemble a complete useful device.

- PWM brightness control
- Light sensor driving LED brightness automatically
- Manual override and auto/manual modes

> Inputs and outputs can be combined into a useful system.

---

## [Week 4 — Make It Smart / Extension Day](Week4/README.md)

**Goal:** Add polish and personalization.

- Non-blocking timing with `millis()`
- Multi-mode behavior and simple state machines
- Extensions: transistor, servo, multiple LEDs, buzzer

> Small changes in software can create much richer behavior.
